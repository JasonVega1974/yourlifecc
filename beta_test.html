📧 Brevo — Email Verification Steps
1. Login to app.brevo.com
2. Go to Transactional → Email
3. Check logs for each test email sent
4. Verify to/from/subject are correct
5. Check Campaigns for any blast emails
6. Verify Contact lists for new signups
7. Check Automations for welcome sequence
8. Monitor bounce/spam rates
💳 Stripe — Payment Verification Steps
1. Login to dashboard.stripe.com → switch to Test Mode
2. Payments → verify test charges went through with card 4242 4242 4242 4242
3. Customers → verify customer record created with correct email
4. Subscriptions → verify plan (monthly/yearly/lifetime) status is active
5. Webhooks → Supabase dashboard endpoint firing correctly
