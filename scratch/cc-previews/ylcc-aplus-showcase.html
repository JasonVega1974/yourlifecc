Child constellation — design-compliant

fixes the 6-color violation
The fix. The current child constellation paints six destination colors on one surface — the exact pattern your visual-design/SKILL.md calls "the classic violation." This rework uses one brass star language for all six domains and spends color only on today's single focus node. Restraint reads as quality: it looks more premium instantly. Why this is the highest-ROI "fun" change. Today, progress is siloed: faith has streaks, traits have XP, each domain has its own badges — a kid earns six kinds of points that never add up. Duolingo's whole moat is the opposite: one streak, one XP total, one daily ring, one league. Tap the tiles — every completion funnels into the same ring + flame + total, with the juice layer (XP-burst, "+N" overshoot, milestone confetti) that makes it feel good. Wire real completions to a single awardXP(n, source) and keep the existing per-domain badges underneath.
